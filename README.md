# BodomCreed

BodomCreed é um laboratório de experimentação e também serve como portfólio pessoal.  
Aqui você encontrará projetos, testes e ideias desenvolvidas para aprendizado, demonstração de habilidades e exploração de novas tecnologias.

Sinta-se à vontade para explorar e acompanhar as atualizações!

---

**Configuração:**  
Crie um arquivo `.env` na raiz do projeto com as variáveis de ambiente necessárias:

```
DJANGO_DEBUG=False
DJANGO_SECRET_KEY=sua_chave_secreta_aqui
```

DJANGO_DEBUG pode ser `True` ou `False` dependendo do ambiente (desenvolvimento ou produção).
